### Biểu Mẫu Dự Báo Bằng Hồi Quy Tuyến Tính

Mô hình hồi quy tuyến tính đơn giản (Simple Linear Regression): Y = a + bX, trong đó Y là biến cần dự báo, X là biến dự báo, a là hệ số góc (intercept), và b là hệ số hồi quy (slope).

> **Bài học:** Hồi quy tuyến tính trong Excel chỉ tốn 5 phút để build. Nhưng 95% thời gian trong quá trình dự báo phải dành cho: kiểm tra data quality, validate assumptions, interpret results in business context, và communicate uncertainty. Model building là phần dễ; judgment là phần khó.

---

**Simple Linear Regression — công thức và xây dựng:**

```
Model: Y = a + bX + ε
  Y: Dependent variable (what we want to forecast) — e.g., Revenue
  X: Independent variable (predictor) — e.g., GDP growth rate
  a: Intercept (value of Y when X = 0)
  b: Slope (change in Y for each 1-unit change in X)
  ε: Error term (random variation not explained by X)

Calculation of b and a:
  b = Σ[(Xi - X̄)(Yi - Ȳ)] / Σ(Xi - X̄)²
  a = Ȳ - b × X̄

Excel: Use =SLOPE(y_range, x_range) and =INTERCEPT(y_range, x_range)
Or: Data → Data Analysis → Regression (full output)
Or: Insert → Chart → Add Trendline → Display equation
```

---

**Step-by-Step Example — dự báo doanh thu F&B:**

```
Historical data (24 months):
  X = Monthly GDP growth index (relative to baseline 100)
  Y = Monthly Revenue (tỷ VNĐ)

Step 1: Calculate means
  X̄ = average GDP index over 24 months = 103.2
  Ȳ = average revenue = 8.5 tỷ

Step 2: Calculate b (slope)
  b = Σ[(Xi - X̄)(Yi - Ȳ)] / Σ(Xi - X̄)²
  Using Excel SLOPE function → b = 0.45

Step 3: Calculate a (intercept)
  a = Ȳ - b × X̄ = 8.5 - 0.45 × 103.2 = -37.94
  
Step 4: Regression equation
  Revenue = -37.94 + 0.45 × GDP_Index

Step 5: Interpret
  For every 1-point increase in GDP index → Revenue increases 0.45 tỷ
  
Step 6: Forecast
  Forecast GDP Index for next year: 106.0 (analyst estimate)
  Forecast Revenue = -37.94 + 0.45 × 106.0 = 9.76 tỷ/month average
  Annual revenue forecast: 9.76 × 12 = ~117 tỷ
```

**Multiple Linear Regression — when one predictor isn't enough:**

```
Model: Y = a + b₁X₁ + b₂X₂ + b₃X₃ + ε

Example for F&B company:
  Y = Revenue
  X₁ = GDP index (macro driver)
  X₂ = Marketing spend (company decision variable)
  X₃ = # Active stores (capacity)

Excel: Data Analysis → Regression → input multiple X columns

Interpretation:
  b₁ = 0.35: for each 1-point GDP increase, revenue increases 0.35 tỷ (holding others constant)
  b₂ = 0.08: each 1 tỷ marketing spend → 0.08 tỷ revenue (holding others constant)
  b₃ = 1.2: each additional active store → 1.2 tỷ additional monthly revenue

Forecast:
  GDP forecast: 106.0
  Planned marketing spend: 1.5 tỷ/month
  Active stores at year-end: 12
  Revenue = a + 0.35(106) + 0.08(1.5) + 1.2(12) = calculate...
```

---

**Regression Output Interpretation — Excel output:**

```
R² (Adjusted): proportion of variance explained
  R² = 0.78 → model explains 78% of revenue variation → reasonably good

Standard Error: average prediction error
  SE = 0.4 tỷ → average forecast is off by ±0.4 tỷ
  Use for confidence interval: Forecast ± 1.96 × SE for 95% CI

P-values for each coefficient:
  p < 0.05: coefficient is statistically significant (relationship is real)
  p > 0.05: may not be significant (relationship could be chance)
  
  Example: b₁ has p = 0.02 (significant); b₂ has p = 0.45 (not significant)
  Implication: marketing spend may not actually explain revenue variation
               in this dataset — investigate why
```

**Forecast Template — standard format:**

```
Monthly Revenue Forecast using Regression:

Model: Revenue = a + b × GDP_Index
  Parameters: a = -37.94, b = 0.45
  R² = 0.78, SE = 0.38 tỷ

Month     | GDP_Index | Point Forecast | 95% CI Lower | 95% CI Upper
----------|-----------|----------------|--------------|-------------
Jan N+1   | 104.8     | 9.22           | 8.47         | 9.97
Feb N+1   | 104.5     | 9.09           | 8.34         | 9.84
Mar N+1   | 105.2     | 9.40           | 8.65         | 10.15
...       | ...       | ...            | ...          | ...
Annual    |           | 117.8 tỷ       | 108.6 tỷ     | 127.0 tỷ

Note: CI = Confidence Interval; 95% CI means 95% probability actual falls within range
```

> **Phân tích sâu:** Prediction intervals vs confidence intervals: two types of uncertainty in regression. Confidence interval captures uncertainty about the average Y for given X. Prediction interval (always wider) captures uncertainty about a specific future observation. For budgeting, prediction interval is correct — we want to know the range of actual revenue next year, not the average of many possible years.

> **Sai lầm phổ biến #1:** Present only point forecast without uncertainty range. "Revenue will be 117 tỷ." This implies false precision. A forecast with R² = 0.78 has real uncertainty. Better: "Base forecast 117 tỷ; range 109–127 tỷ (95% confidence interval)." This shapes contingency planning.

> **Sai lầm phổ biến #2:** Extrapolate far beyond historical data range. Model trained on GDP index 98–108; then forecast at GDP index 115 (outside historical range). The relationship may not hold at extremes — non-linearity may emerge. Always note when forecasting outside historical range and flag higher uncertainty.

> **Cạm bẫy:** Trust regression without residual analysis. Residuals (actual - predicted) should look random. If residuals show pattern (e.g., always positive in Q1, always negative in Q2), it means the model is missing a seasonal effect — and forecasts will be systematically biased for specific periods.
