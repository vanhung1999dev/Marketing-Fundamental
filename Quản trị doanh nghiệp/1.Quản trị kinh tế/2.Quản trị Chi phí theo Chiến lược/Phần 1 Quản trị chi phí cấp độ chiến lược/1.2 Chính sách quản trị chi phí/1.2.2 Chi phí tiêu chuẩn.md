### Chi Phí Tiêu Chuẩn (Standard Costing)

Standard costing là hệ thống xác định **mức chi phí "nên là"** cho từng đơn vị sản phẩm/dịch vụ, sau đó so sánh với chi phí thực tế để phát hiện sai lệch và nguyên nhân.

```
Chi phí tiêu chuẩn = Standard Material + Standard Labour + Standard Overhead
Variance = Chi phí thực tế − Chi phí tiêu chuẩn
```

> **Bài học:** Standard costing trả lời "chúng ta đang vận hành đúng kế hoạch không?" trong khi activity-based costing trả lời "chúng ta đang assign cost đúng không?" Hai câu hỏi khác nhau, cần cả hai để quản trị chi phí toàn diện.

---

**Xây dựng Standard Cost:**

**1. Standard Material Cost:**
```
= Standard Quantity per Unit × Standard Price per Unit

Ví dụ sản xuất bánh mì:
  Bột mì: 0.5 kg/ổ × 15,000 VNĐ/kg = 7,500 VNĐ/ổ
  Trứng: 0.1 quả/ổ × 3,000 VNĐ/quả = 300 VNĐ/ổ
  ...
  Standard Material = 12,000 VNĐ/ổ
```

**2. Standard Labour Cost:**
```
= Standard Hours per Unit × Standard Rate per Hour

Ví dụ: 0.05 giờ/ổ × 60,000 VNĐ/giờ = 3,000 VNĐ/ổ
```

**3. Standard Overhead:**
```
= Budgeted Overhead / Budgeted Output Level
= Overhead Rate × Standard Hours per Unit
```

---

**Variance Analysis — phân tích sai lệch:**

Variance là chênh lệch giữa actual và standard. Phân tách để biết nguồn gốc:

**Material Variance:**
```
Material Price Variance (MPV)  = (Standard Price − Actual Price) × Actual Quantity
  → Procurement có mua đúng giá không?

Material Usage Variance (MUV)  = (Standard Qty − Actual Qty) × Standard Price
  → Production có dùng đúng định mức không?

Total Material Variance = MPV + MUV
```

**Labour Variance:**
```
Labour Rate Variance (LRV)       = (Standard Rate − Actual Rate) × Actual Hours
  → HR có maintain wage structure không?

Labour Efficiency Variance (LEV) = (Standard Hours − Actual Hours) × Standard Rate
  → Workers có làm việc đúng năng suất không?
```

---

**Ví dụ thực tế — xưởng may VN:**

Standard cost một áo sơ mi:
- Vải: 2m × 80,000 = 160,000 VNĐ
- Chỉ/phụ liệu: 15,000 VNĐ
- Lao động: 0.5h × 50,000/h = 25,000 VNĐ
- **Tổng Standard: 200,000 VNĐ/cái**

Actual tháng này (500 áo):
- Vải thực: 1,050m × 85,000 = 89,250,000 VNĐ (vs standard: 500×2×80,000 = 80,000,000)
- Lao động thực: 280h × 52,000 = 14,560,000 VNĐ (vs standard: 250h × 50,000 = 12,500,000)

Variance phân tích:
| Variance | Tính toán | Kết quả | Nguyên nhân cần điều tra |
|---|---|---|---|
| Material Price | (80,000−85,000)×1,050 | −5,250,000 | Giá vải tăng hay mua sai supplier? |
| Material Usage | (1,000−1,050)×80,000 | −4,000,000 | Vải bị lãng phí hay cắt pattern sai? |
| Labour Rate | (50,000−52,000)×280 | −560,000 | Tăng lương hay overtime nhiều hơn? |
| Labour Efficiency | (250−280)×50,000 | −1,500,000 | Năng suất thấp — đào tạo? máy móc? |

---

**Khi nào Standard Costing phù hợp:**

| Phù hợp | Không phù hợp |
|---|---|
| Sản xuất hàng loạt với process lặp đi lặp lại | Custom/project work — mỗi order khác nhau |
| Stable input costs (có thể forecast) | Highly volatile commodity prices |
| Mature processes với historical data | New products/processes — no baseline |
| Manufacturing, F&B, pharma | Service industries (difficult to standardize) |

---

**Để Standard Costing work:**

```
1. Standards phải realistic — không quá chặt (không đạt được) hay quá lỏng (không có value)
2. Update định kỳ — nếu không, standards trở nên stale và variances meaningless
3. Variance investigation protocol: chỉ investigate khi variance > threshold (ví dụ: >5% hoặc >10 triệu)
4. Accountability: ai owns which variance?
   - Material Price → Procurement Manager
   - Material Usage → Production Manager
   - Labour Efficiency → Line Supervisor
```

> **Phân tích sâu:** Toyota không dùng traditional standard costing — họ dùng "target costing" (bắt đầu từ target price và work backward). Nhưng nhiều VN manufacturers thấy standard costing là foundation cần thiết trước khi upgrade lên kaizen/target costing. Key insight: standard costing không phải goal — nó là tool để create cost transparency, which enables improvement.

> **Sai lầm phổ biến #1:** Set standards một lần rồi dùng mãi. Industry material prices, labour rates, và efficiency benchmarks đổi theo năm. Outdated standards → variances meaningless → system becomes noise, not signal. Review standards ít nhất annually, ideally semi-annually.

> **Sai lầm phổ biến #2:** Report variances nhưng không investigate root cause và take action. "Unfavorable variance" là symptom — cần diagnose và treat. Nếu variance reports chỉ nằm trong file mà không trigger action → system waste of time.

> **Cạm bẫy:** Standard costing có thể create wrong incentives. If production manager is measured on labour efficiency variance → may push workers too hard → quality issues → customer complaints → more cost down the line. Balanced scorecard: efficiency AND quality AND delivery.
